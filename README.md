# PythonStock

A basic stock analysis program

Intakes CSV data, during which it cleans it and organizes it for metrics to be taken
Daily returns, volatility, moving average, and drawdown are calculated over a 20/50 day period.
Then based on the data, it is analyzed and insights are generated dependant on said metrics. Finally being outputted.


This project is done as of 2026-01-16, any further changes that are finished will be documented below.
